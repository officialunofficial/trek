We introduce the Darwin Gödel Machine (DGM), an AI system capable of rewriting its own code to improve itself over time.

Unlike previous self-improving systems, the DGM uses an evolutionary approach inspired by both Darwin and Gödel to iterate on its own architecture.

The system has shown remarkable results on a range of benchmarks, demonstrating the viability of recursive self-improvement in practical AI systems.
